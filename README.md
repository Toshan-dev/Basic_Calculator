# 🧮 Terminal Loop Calculator or Basic calculator

A robust, terminal-based calculator built in Python that executes arithmetic operations inside a continuous execution loop. 

## 🚀 Features
* **Continuous Execution:** Uses a `while True` loop so the program stays open for multiple calculations without crashing or closing.
* **Input Validation:** Automatically processes integer and floating-point inputs.
* **Exception Guard:** Includes built-in protection against `ZeroDivisionError` anomalies to gracefully prevent crashes if a user inputs zero as a denominator.
* **Error Handling:** Identifies and handles invalid operator selections automatically.

## 🛠️ How to Run Locally

1. Make sure you have Python installed on your PC.
2. Clone or download this repository.
3. Open your terminal in the project folder and run:
   ```bash
   python calculator.py
